# badge-print — печать бейджей участников конференции

Простая веб-страница для печати именных этикеток (бейджей) на **мобильный термопринтер** прямо с телефона. Никаких приложений и установки — один HTML-файл, работает в браузере.

🔗 **Открыть:** https://dimakotov1c.github.io/badge-print/

---

## Что умеет

- 📋 **Загрузка списка** участников из CSV-файла или вставкой таблицы из Excel/1С.
- 🔤 **Авто-определение** разделителя (`,` / `;` / табуляция) и кодировки (**UTF-8 / Windows-1251**) — кириллица не «ломается».
- 🏷️ **Этикетка по центру:** крупно «Имя Фамилия», ниже — Компания и Должность.
- 📐 **Настраиваемый размер** этикетки и шрифта (по умолчанию **100×50 мм**).
- 👀 **Просмотр по одной** этикетке с навигацией «Назад / Вперёд» и счётчиком.
- 📤 **Печать картинкой** — этикетка формируется в PNG (≈203 dpi) и отправляется через меню «Поделиться» в приложение принтера. Есть поворот на 0/90/180/270°.
- 🖨️ Запасной вариант — обычная печать через **AirPrint** и сохранение картинки в Фото/Файлы.

---

## Формат данных

Колонки по порядку: **Имя, Фамилия, Компания, Должность**. Первая строка — заголовки.

```csv
Имя,Фамилия,Компания,Должность
Иван,Иванов,ООО «Рога и копыта»,Менеджер по продажам
Анна,Петрова,ООО "Буду делать",Финансовый директор
Пётр,Сидоров,ИП Сидоров,Разработчик
```

Кавычки-«ёлочки» `«»` и прямые `"…"` внутри названий обрабатываются корректно.

## Как пользоваться

1. Откройте страницу на телефоне.
2. Нажмите **«Загрузить пример данных»** для проверки или выберите свой CSV.
3. При необходимости подправьте размер этикетки/шрифта и поворот картинки.
4. Пролистайте до нужного участника → **«Отправить на принтер (картинкой)»** → в меню «Поделиться» выберите приложение термопринтера.
5. Нажмите **«Вперёд ›»** и распечатайте следующую.

## Печать на iPhone

iPhone не печатает на большинство мобильных термопринтеров через AirPrint, поэтому этикетка отправляется **картинкой** в фирменное приложение принтера через стандартное меню «Поделиться». Убедитесь, что приложение принтера установлено из App Store.

Размер этикетки можно задать на странице; при выборе носителя в приложении принтера укажите тот же размер (по умолчанию 100×50 мм).

## Технические детали

- Один самодостаточный файл `index.html` — без сторонних библиотек и без обращений в интернет.
- Данные участников вводятся и обрабатываются **только на устройстве**, никуда не передаются.
- Картинка этикетки рисуется на `<canvas>` в разрешении термопринтера (8 точек/мм ≈ 203 dpi).
- Размещение: GitHub Pages (ветка `main`, корень репозитория).

## Обновление страницы

Изменить сайт = заменить файл `index.html` в репозитории:
**Add file → Upload files →** перетащить новый `index.html` **→ Commit changes**.
Сайт обновится автоматически через ~1 минуту.
